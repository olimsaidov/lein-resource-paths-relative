# lein-resource-paths-relative

## Usage

Add `lein-resource-paths-relative` to `:plugins` in `project.clj`, e.g.

    :plugins [[lein-resource-paths-relative "0.0.1"]]

Add relative resources paths:

    :resource-paths-relative [["foo" "foo/bar/*"]]
